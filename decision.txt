[dɪˈsɪʒən] - решение

Как правильно решать задачи

Внимательно прочитать задачу и выписать условия
Декомпозировать задачу
Написать последовательный псевдокод


Понять суть задачи ( вдумчиво вслух проговори задачу  )
Декомпозиция ( разделение целого на части  )
Не думай : что должна делать программа
Думай : чтобы сделал человек
Набросать псевдокод ( описание действий пошагово на русском  )

Какие входные данные
Какие данные на выходе
Следить за названием переменных ( Осмысленные и читаемые )
Какие ошибки и проблемы могут возникнуть? ( Что делать, если ?  )
    Придумать как можно больше ответить на них  и реализовать программно
Имея уже работающее решение подумать как его можно улучшить ( провести рефакторинг )
    Можно ли получить этот результат  как -то иначе ?
    Понятно ли это решение с первого взгляда ?
    Можно ли улучшить производительность решения  ?
    Как эту задачу решают другие люди ?

ВАЖНО : Никогда не начинай решение задачи с кодинга без декомпозиции
Смотри на другие решения  \  Читай чужой код
